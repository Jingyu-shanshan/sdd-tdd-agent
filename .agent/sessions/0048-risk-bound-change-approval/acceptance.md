# Acceptance

- [x] Risk is deterministic and based only on validated changed paths/kinds.
- [x] Medium/high changes cannot proceed without explicit human approval.
- [x] Approval is bound to one active Session and exact change digest.

- [x] Local and hosted quality gates pass.
