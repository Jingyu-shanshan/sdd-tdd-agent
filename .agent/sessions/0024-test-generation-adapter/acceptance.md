# Acceptance criteria

- [x] JSON exchange sends exactly all typed request fields.
- [x] Valid nested output decodes into immutable plan/case models.
- [x] Every structural type and exact key set is enforced.
- [x] Codex receives a strict nested Schema with no additional properties.
- [x] Codex execution is ephemeral, read-only, shell-free, and workspace-scoped.
- [x] Temporary exchange content is cleaned.
- [x] Errors expose no sensitive request or process content.
- [x] Existing semantic workflow validation remains authoritative.
- [x] No dependency or unnecessary `.gitignore` update is introduced.
