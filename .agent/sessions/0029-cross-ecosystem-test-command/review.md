# Review

Detection is read-only and returns token tuples only. It never downloads tools,
runs a process, or invokes a shell. Java commands require verified JUnit 5 and
standard test roots. Node commands require a non-empty project test script,
consistent package-manager evidence, and matching framework dependencies and
script tokens.

Node name filters are `re.escape`-protected and anchored, while file arguments
remain exact validated relative paths. Invalid JSON, duplicate keys, ambiguous
locks, conflicting metadata, incomplete Angular evidence, multiple frameworks,
unsafe selectors, and mixed Java build tools fail explicitly.

No dependency, configuration mutation, secret, unrelated API change, or new
ignore pattern was introduced.
