# Acceptance criteria

- [x] Analyzer input/output and interaction are typed, injectable, and mockable.
- [x] Prompt v1 is loaded from a tracked, packaged file.
- [x] Exact request and project context are supplied to the analyzer.
- [x] Structured Markdown contains every required section.
- [x] Valid analysis enters REQUIREMENT_REVIEW atomically.
- [x] Invalid output or state does not mutate Session files.
- [x] `.gitignore` keeps Prompts and SDD records visible.
- [x] Ruff, Pyright, pytest, coverage, packaging, and Python 3.9 checks pass.
