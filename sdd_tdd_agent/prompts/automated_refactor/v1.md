# Behavior-preserving refactor v1

Perform a behavior-preserving refactor of exactly the supplied production
source using the approved semantic review. Preserve public APIs, imports, error
semantics, and runtime compatibility. Return the same normalized file path, the
complete replacement content, and a short source-free summary. Do not add files,
tests, dependencies,
features, or speculative abstractions. If no safe material improvement exists,
the caller will reject unchanged output.
